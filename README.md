This project is a web-based Movie Recommendation System that allows users to input the name of a movie and receive suggestions for similar titles. The backend, built with Flask and powered by TF-IDF Vectorization and Cosine Similarity, processes movie metadata to generate recommendations. The frontend, developed using React and Framer Motion, provides a smooth and interactive user experience with animated swipeable cards displaying recommended movies. This project demonstrates full-stack development skills, combining machine learning-based logic with a modern UI.
